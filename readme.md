# NearbyStay

![](https://media.giphy.com/media/vODgBalDIUxbvcBu3E/giphy.gif)

### Description
Carousel component for room reservation service

### Requirement
- npm v8
- MongoDB v4.2

### Installation

1. Install npm
```sh
npm install
```

2. Setup MongoDB
```sh
npm run db:setup
```

3. Run Webpack to bundle.
```sh
npm run webpack
```

4. Start server
```sh
npm start
```
